# Calculadora de Estadísticas

Este es un proyecto en Java que calcula estadísticas (Promedio, Mediana y Moda) a partir de un archivo CSV. Está diseñado como un ejercicio para aplicar los principios de diseño SOLID, utilizando el Patrón Strategy.

El programa se ejecuta desde la línea de comandos, aceptando un archivo CSV como entrada y banderas para especificar qué cálculos realizar.

## Prerrequisitos

Para compilar y ejecutar este proyecto, necesitarás:

* Java 17+
* Apache Maven 3.8+
* Bash (para usar el script de ejecución)

---

## Cómo Usarlo

El proyecto se ejecuta usando el script de Bash `run.sh`. Este script se encarga de compilar el proyecto con Maven, empaquetarlo en un JAR ejecutable (incluyendo todas las dependencias) y luego ejecutarlo con los argumentos que le pases.

### Sintaxis

```bash
./run.sh <ruta_del_csv> [banderas...]
